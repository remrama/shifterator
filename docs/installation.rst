Installation
============

Shifterator requires Python 3.8 or higher.

The following packages are also required:

- `Matplotlib <https://matplotlib.org/index.html>`_
- `NumPy <https://numpy.org/>`_

Install from GitHub
-------------------

.. important::

    The version of Shifterator on PyPI (``0.3.0``) is outdated and no longer maintained.
    Please install directly from GitHub.

The latest version of :code:`shifterator` can be installed directly from GitHub:

.. code-block:: bash

    pip install git+https://github.com/ryanjgallagher/shifterator.git

Install Development Version
---------------------------

To install a local editable copy for development:

.. code-block:: bash

    git clone https://github.com/ryanjgallagher/shifterator.git
    cd shifterator
    pip install -e .
